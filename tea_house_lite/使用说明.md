## 📖 部署步骤

### 第一步：注册 Hugging Face

1. 访问：https://huggingface.co/join
2. 填写用户名、邮箱、密码
3. 验证邮箱（检查垃圾箱）
4. （可选）验证手机号

### 第二步：创建 Space

1. 访问：https://huggingface.co/new-space
2. 填写信息：
   - **Owner**：选择您的用户名
   - **Space name**：tea-house-system
   - **License**：MIT
   - **Select the SDK**：**Streamlit**（重要！）
   - **Public/Private**：Public
3. 点击 **"Create Space"**

### 第三步：上传文件

1. 在您的 Space 页面，点击右侧 **"Files and versions"**
2. 点击 **"Add file"** → **"Upload files"**
3. 依次上传3个文件：
   - 选择 `app.py` → 点击 **"Commit changes to main"**
   - 选择 `requirements.txt` → 点击 **"Commit changes to main"**
   - 选择 `README.md` → 点击 **"Commit changes to main"**

### 第四步：等待部署

- 等待 2-5 分钟
- 页面显示 **"Running"** 即成功

### 第五步：访问系统

- 访问：`https://您的用户名-tea-house-system.hf.space`
- 开始使用！
