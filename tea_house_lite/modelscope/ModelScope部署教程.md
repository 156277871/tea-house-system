# 🚀 ModelScope（魔搭社区）部署教程

**国内免费部署茶楼管理系统，无需VPN，无需GitHub！**

---

## 📋 准备工作

您需要准备：
- ✅ 一个邮箱（QQ、163、Gmail 都可以）
- ✅ 支付宝或微信账号（用于实名认证）
- ✅ 手机号（用于验证）

---

## 📝 第一步：注册 ModelScope 账号

### 1.1 打开注册页面
访问：https://modelscope.cn

### 1.2 点击注册
- 点击右上角的"登录/注册"
- 选择"注册账号"

### 1.3 填写注册信息
- **手机号**：输入您的手机号
- **验证码**：点击"获取验证码"，输入收到的短信验证码
- **密码**：设置一个安全的密码
- **确认密码**：再次输入密码
- 勾选"同意用户协议"
- 点击"注册"

### 1.4 登录账号
- 使用手机号和密码登录

---

## 🔐 第二步：实名认证（必须完成！）

### 2.1 进入个人中心
- 点击右上角头像
- 选择"个人中心"

### 2.2 开始实名认证
- 找到"实名认证"选项
- 点击"去认证"

### 2.3 选择认证方式
- **支付宝认证**（推荐）：扫码认证，最快
- **微信认证**：扫码认证
- **银行卡认证**：填写银行卡信息

### 2.4 完成认证
- 按照提示完成认证
- 一般1-5分钟即可完成

**注意：** 不完成实名认证无法创建 Space！

---

## 🆕 第三步：创建 Space

### 3.1 进入创建页面
- 登录后，点击顶部导航的"模型"
- 或者直接访问：https://modelscope.cn/my/myapplications

### 3.2 创建新的 Space
- 点击"创建应用"或"创建 Space"
- 或者访问：https://modelscope.cn/studio

### 3.3 填写 Space 信息

**应用名称（Space Name）**：
- 输入：tea-house-system
- 或者输入您喜欢的英文名称（只能用字母、数字、短横线）

**可见性**：
- 选择：公开（Public）

**应用类型（App Type）**：
- 选择：Space

**SDK**：
- **重要！** 选择：Streamlit

### 3.4 创建 Space
点击页面底部的 **"创建"** 按钮

---

## 📤 第四步：上传文件

### 4.1 下载文件包
从对象存储下载以下3个文件：
- `app.py` - 主程序文件
- `requirements.txt` - 依赖文件
- `README.md` - 说明文件

### 4.2 打开您创建的 Space
创建成功后会自动跳转到您的 Space，或者访问：
```
https://modelscope.cn/studio/您的用户名/tea-house-system
```

### 4.3 上传文件

**方法一：网页上传（推荐）**

1. 点击右侧的 **"文件"** 或 **"Files"** 标签
2. 点击 **"上传文件"** 或 **"Upload"**
3. 依次上传3个文件：
   - 点击 **"选择文件"**
   - 选择电脑上的 `app.py` 文件
   - 点击 **"提交"** 或 **"Commit"**
4. 重复步骤3，上传 `requirements.txt` 和 `README.md`

**方法二：批量上传**

1. 点击右侧的 **"文件"** 标签
2. 点击 **"上传文件"**
3. **一次性选择所有3个文件**（按住Ctrl键多选）
4. 点击 **"提交"**

---

## ⏳ 第五步：等待部署完成

### 5.1 自动构建
- 文件上传后，ModelScope 会自动开始构建
- 这需要 **5-10分钟**，请耐心等待

### 5.2 查看构建状态
- 在 **"应用"** 或 **"App"** 标签页可以看到构建日志
- 状态显示 **"运行中"** 即表示部署成功
- 如果有错误，会在日志中显示

### 5.3 可能遇到的问题

**构建失败？**
- 检查文件名是否正确（app.py、requirements.txt、README.md）
- 确保上传了所有3个文件
- 查看 **"日志"** 或 **"Logs"** 标签中的错误信息
- 删除 Space 重新创建

**构建时间太长？**
- 正常现象，首次构建需要5-10分钟
- 后续更新会更快

---

## 🎉 第六步：开始使用

### 6.1 访问您的系统
部署成功后，访问：
```
https://modelscope.cn/studio/您的用户名/tea-house-system
```

### 6.2 初始化数据

按照以下顺序创建基础数据：

**步骤1：创建门店**
- 点击左侧菜单 **"🏪 门店管理"**
- 切换到 **"新增门店"** 标签
- 填写：
  - 门店名称：例如"北京朝阳店"
  - 门店编码：例如"BJ001"
  - 地址、电话（选填）
- 点击 **"创建"**

**步骤2：创建商品**
- 点击左侧菜单 **"🛍️ 商品管理"**
- 切换到 **"新增商品"** 标签
- 填写：
  - 商品名称：例如"龙井茶"
  - 商品编码：例如"P001"
  - 分类：选择（茶叶/茶具/点心/饮品）
  - 单价：例如 128
  - 单位：例如 "盒"
- 点击 **"创建"**

**步骤3：创建员工**
- 点击左侧菜单 **"👥 员工管理"**
- 切换到 **"新增员工"** 标签
- 填写：
  - 姓名：例如"张三"
  - 电话：例如"13800138000"
  - 职位：选择（店长/店员/收银员）
  - 所属门店：选择刚才创建的门店
- 点击 **"创建"**

**步骤4：创建会员**
- 点击左侧菜单 **"💎 会员管理"**
- 切换到 **"新增会员"** 标签
- 填写：
  - 姓名：例如"李四"
  - 电话：例如"13900139000"
- 点击 **"创建"**

**步骤5：库存入库**
- 点击左侧菜单 **"📦 库存管理"**
- 切换到 **"库存入库"** 标签
- 选择门店和商品
- 输入入库数量
- 点击 **"入库"**

**步骤6：创建订单**
- 点击左侧菜单 **"📝 订单管理"**
- 切换到 **"创建订单"** 标签
- 选择门店、支付方式、商品和数量
- 点击 **"创建"**

## 🆘 常见问题

### Q1: 注册时收不到验证码？
**A:** 检查手机号是否正确，确保信号良好，稍后再试

### Q2: 实名认证很麻烦？
**A:** 只需要认证一次，推荐使用支付宝扫码，最快1分钟完成

### Q3: 上传文件后一直显示"构建中"？
**A:** 正常现象，首次构建需要5-10分钟，请耐心等待

### Q4: 访问页面时显示 404 错误？
**A:** 等待部署完成后再访问，检查URL是否正确

### Q5: 数据会丢失吗？
**A:** 不会！ModelScope 提供数据持久化

### Q6: 可以让多个员工同时使用吗？
**A:** 可以，分享URL给员工即可

### Q7: 可以手机上使用吗？
**A:** 可以，在手机浏览器打开URL即可

### Q8: 部署失败怎么办？
**A:**
1. 检查是否完成了实名认证
2. 检查文件名是否正确
3. 查看日志中的错误信息
4. 删除 Space 重新创建
