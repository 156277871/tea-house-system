# 🏪 连锁茶楼管理系统

基于 Streamlit 的轻量级连锁茶楼管理系统，**完全免费！**

## ✨ 功能特性

- 📊 控制台 - 实时监控营业数据
- 🏪 门店管理 - 创建和管理门店信息
- 👥 员工管理 - 员工档案管理
- 💎 会员管理 - 会员信息与充值
- 🛍️ 商品管理 - 商品分类管理
- 📦 库存管理 - 库存监控与入库
- 📝 订单管理 - 订单创建与查询
- 💰 财务报表 - 数据分析

## 🚀 部署到 Streamlit Cloud（免费）

### 步骤1：注册 GitHub
1. 访问 https://github.com
2. 点击 "Sign up" 注册
3. 完成邮箱验证

### 步骤2：创建仓库
1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 仓库名：tea-house-lite
4. 选 "Public"（免费）
5. 点击 "Create repository"

### 步骤3：上传代码
1. 点击 "uploading an existing file"
2. 上传这3个文件：
   - app.py
   - requirements.txt
   - README.md
3. 点击 "Commit changes"

### 步骤4：部署
1. 访问 https://share.streamlit.io
2. 用 GitHub 登录
3. 点击 "New app"
4. 选择仓库：tea-house-lite
5. 分支：main
6. 文件：app.py
7. 点击 "Deploy"

### 步骤5：使用
- 等待2-5分钟
- 访问页面显示的URL
- 开始使用！
