# 连锁茶楼管理系统 V4.1

一个基于Streamlit的轻量级茶楼管理系统，支持多门店、桌台、会员、订单、库存等完整业务流程。

## ✨ 主要功能

- 📊 **控制台** - 实时查看营业数据、台位状态
- 🎯 **经营管理** - 桌台开台、点单、结账完整流程
- ⚙️ **系统设置**
  - 🏪 门店管理
  - 🪑 桌台管理
  - 👥 员工管理
  - 🛍️ 商品管理
  - 📦 库存管理
- 💎 **会员管理** - 会员信息、等级管理
- 📝 **订单管理** - 订单查询、历史记录
- 💰 **财务报表** - 营业额统计、台位统计

## 🚀 快速开始

### 环境要求

- Python 3.7+
- pip

### 安装步骤

1. 克隆项目
```bash
git clone <repository-url>
cd tea_house_lite
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行应用
```bash
streamlit run app.py
```

4. 访问应用

打开浏览器访问：http://localhost:8501

## 📖 使用说明

### 首次使用流程

1. **创建门店** - 进入「设置」→「门店管理」，创建第一个门店
2. **创建桌台** - 进入「设置」→「桌台管理」，为门店添加桌台
3. **创建商品** - 进入「设置」→「商品管理」，添加可销售商品
4. **库存入库** - 进入「设置」→「库存管理」，为商品添加库存
5. **开始营业** - 进入「经营管理」，选择门店，开始开台、点单、结账

### 基本操作

- **开台**: 选择空闲桌台 → 点击「开台」→ 选择会员（可选）
- **点单**: 在点单标签页 → 选择商品 → 输入数量 → 点击「点单」
- **结账**: 在结账标签页 → 选择支付方式 → 确认金额 → 点击「确认结账」

## 🔧 配置说明

### 数据库

系统使用SQLite数据库，数据文件 `tea_house.db` 会自动在项目根目录创建。

### 端口配置

默认端口为 `8501`，可通过以下命令修改：

```bash
streamlit run app.py --server.port=8502
```
