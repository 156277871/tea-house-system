# 🏪 连锁茶楼管理系统

基于 Streamlit 的轻量级连锁茶楼管理系统，支持6家门店的统一管理。

## ✨ 功能特性

- 📊 **控制台** - 实时监控营业数据
- 🏪 **门店管理** - 创建和管理门店信息
- 👥 **员工管理** - 员工档案管理
- 💎 **会员管理** - 会员信息与充值
- 🛍️ **商品管理** - 商品分类管理
- 📦 **库存管理** - 库存监控与入库
- 📝 **订单管理** - 订单创建与查询
- 💰 **财务报表** - 数据分析与导出

## 🚀 部署到 Streamlit Cloud（完全免费）

### 步骤1：注册 GitHub 账号
1. 访问 https://github.com
2. 点击 "Sign up" 注册
3. 完成邮箱验证

### 步骤2：创建 GitHub 仓库
1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 仓库名称：tea-house-lite
4. 选择 "Public"（免费）
5. 点击 "Create repository"

### 步骤3：上传代码
1. 在新创建的仓库页面，点击 "uploading an existing file"
2. 将这3个文件拖拽到上传区域：
   - app.py
   - requirements.txt
   - README.md
3. 点击 "Commit changes"

### 步骤4：部署到 Streamlit Cloud
1. 访问 https://share.streamlit.io
2. 点击 "Sign in" 并使用 GitHub 登录
3. 点击 "New app"
4. 选择你的仓库：tea-house-lite
5. 选择分支：main 或 master
6. 主文件路径：app.py
7. 点击 "Deploy"

### 步骤5：访问系统
- 等待 2-5 分钟部署完成
- 访问你的应用地址（显示在页面上）
- 开始使用！

## 📝 使用指南

### 第一次使用顺序
1. **创建门店** → 进入"门店管理" → "新增门店"
2. **创建商品** → 进入"商品管理" → "新增商品"
3. **创建员工** → 进入"员工管理" → "新增员工"
4. **创建会员** → 进入"会员管理" → "新增会员"
5. **库存入库** → 进入"库存管理" → "库存入库"
6. **创建订单** → 进入"订单管理" → "创建订单"

### 查看数据
- **控制台**：查看今日营业数据
- **财务报表**：查看日报、月报、门店对比
