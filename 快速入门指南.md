# 🏪 连锁茶楼管理系统 - 快速入门指南

## 🚀 第一步：启动系统

### Windows 用户
1. 找到 `start_web.bat` 文件
2. 双击运行
3. 等待几秒钟，浏览器会自动打开

### Mac/Linux 用户
1. 打开终端
2. 运行命令：
   ```bash
   chmod +x start_web.sh
   ./start_web.sh
   ```
3. 浏览器自动打开，访问 `http://localhost:8501`

---

## 📝 第二步：初始化系统数据

### 2.1 创建门店
1. 点击左侧菜单的「🏪 门店管理」
2. 点击「新增门店」标签
3. 填写信息（所有带 * 的为必填项）：
   - 门店名称：西湖茶楼
   - 门店编码：ST001
   - 门店地址：杭州市西湖区
   - 联系电话：0571-12345678
4. 点击「创建门店」按钮
5. ✅ 门店创建成功！

**建议创建6家门店**：
- ST001 - 西湖茶楼
- ST002 - 东湖茶楼
- ST003 - 南湖茶楼
- ST004 - 北湖茶楼
- ST005 - 中河茶楼
- ST006 - 钱塘茶楼

### 2.2 创建商品
1. 点击左侧菜单的「🛍️ 商品管理」
2. 点击「新增商品」标签
3. 填写信息：
   - 商品名称：龙井茶
   - 商品编码：P001
   - 商品分类：茶叶
   - 单价：80
   - 单位：克
4. 点击「创建商品」按钮

**建议创建以下商品**：
| 编码 | 名称 | 分类 | 单价(元) | 单位 |
|------|------|------|----------|------|
| P001 | 龙井茶 | 茶叶 | 80 | 克 |
| P002 | 普洱茶 | 茶叶 | 60 | 克 |
| P003 | 铁观音 | 茶叶 | 70 | 克 |
| P004 | 红茶 | 茶叶 | 50 | 克 |

### 2.3 创建员工
1. 点击左侧菜单的「👥 员工管理」
2. 点击「新增员工」标签
3. 填写信息：
   - 员工姓名：张三
   - 联系电话：13800138000
   - 职位：店长
   - 所属门店：选择刚创建的门店
4. 点击「创建员工」按钮

### 2.4 创建会员
1. 点击左侧菜单的「💎 会员管理」
2. 点击「新增会员」标签
3. 填写信息：
   - 会员姓名：李四
   - 手机号：13900139000
4. 点击「创建会员」按钮

### 2.5 库存入库
1. 点击左侧菜单的「📦 库存管理」
2. 点击「库存入库」标签
3. 填写信息：
   - 门店：选择门店
   - 商品：选择商品
   - 入库数量：500
4. 点击「确认入库」按钮

## 💰 第三步：开始营业

### 创建第一个订单
1. 点击左侧菜单的「📝 订单管理」
2. 点击「创建订单」标签
3. 填写信息：
   - 门店：选择门店
   - 会员：选择会员（可选）
   - 支付方式：微信支付
4. 选择商品：
   - 商品1：选择龙井茶
   - 数量：10
5. 点击「创建订单」按钮
6. ✅ 订单创建并支付成功！

## 📊 第四步：查看数据

### 查看今日营业情况
1. 点击左侧菜单的「📊 控制台」
2. 即可看到：
   - 今日订单数
   - 今日营业额
   - 门店排名
   - 会员统计
   - 最近订单

### 查看财务报表
1. 点击左侧菜单的「💰 财务报表」
2. 选择「日报表」、「月报表」或「门店对比」
3. 选择日期和门店
4. 点击「查询」查看数据

---

## 🎯 常用操作

### 会员充值
1. 点击「💎 会员管理」→「会员充值」标签
2. 选择会员
3. 输入充值金额
4. 点击「确认充值」

### 查看库存
1. 点击「📦 库存管理」→「库存查询」标签
2. 选择门店
3. 勾选「仅显示预警商品」可以查看库存不足的商品

### 查询订单
1. 点击「📝 订单管理」→「订单列表」标签
2. 可以按门店和状态筛选订单

---

## 💡 小技巧

1. **批量创建**：可以多次点击「新增」按钮，连续创建多条数据
2. **搜索功能**：在列表页面可以使用搜索框快速查找数据
3. **刷新数据**：操作完成后会自动刷新列表
4. **数据统计**：控制台可以快速查看今日营业情况

## ❓ 遇到问题？

### 问题1：页面加载慢
**解决**：检查网络连接，或刷新页面

### 问题2：数据保存失败
**解决**：检查必填项是否填写完整，编码是否重复

### 问题3：找不到刚创建的数据
**解决**：下拉列表页查看更多数据，或使用搜索功能

### 问题4：订单创建失败
**解决**：确保商品已入库，库存充足
